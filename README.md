# Bilibili Downloader
这是一个用于下载Bilibili 视频Python 脚本。 2024-12-10

## 功能
- 从 Bilibili 获取视频和音频并下载链接
- 使用 `ffmpeg` 合并视频和音频
- 生成合并后的 MP4 文件

## 安装依赖

在项目根目录下运行以下命令安装所有必需的 Python 包：

```bash
pip install -r requirements.txt

## 运行
python bilibiliDownloader.py
